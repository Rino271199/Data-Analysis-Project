# 🏏 IPL Data Analytics Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard built to analyze Indian Premier League (IPL) data across multiple seasons. It provides insights into team performance, player statistics, batting and bowling records, match outcomes, and season highlights using interactive visualizations.

---

## 🎯 Objectives
- Analyze IPL performance across seasons.
- Identify top-performing teams and players.
- Compare batting and bowling statistics.
- Build an interactive dashboard for decision-making.

---

## 🛠️ Tools & Technologies
- Power BI Desktop
- Power Query
- DAX
- SQL (data preparation)
- Excel / CSV

---

## 📂 Dataset
The project uses IPL datasets containing:
- Match details
- Ball-by-ball data
- Teams
- Players

---

## 📊 Dashboard Features
- Season selector
- Tournament winner
- Orange Cap & Purple Cap
- Team-wise performance
- Runs, wickets, fours & sixes
- Batting and bowling analysis
- Interactive slicers and cross-filtering

---

## 📈 KPIs
- Total Matches
- Total Runs
- Total Wickets
- Total Fours
- Total Sixes
- Highest Run Scorer
- Highest Wicket Taker
